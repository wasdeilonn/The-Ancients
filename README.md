# The Ancients
A mod that brings the long awaited Ancients tribe to Polytopia!


# Done features:
- lightning rod that gets struck with lightning each start of the turn. each lightning strike can either power nearby electrodes and batteries, charge a unit with the capacitor ability, or release an explosion dealing a lot of damage to all units nearby, even allied ones.
- electrode improvement, that makes 1 star per lightning strike
- accumulator unit, that can be charged by lightning warriors, archers and riders, and can release an explosion, that gets stronger the more charges the unit has (max 3 for a 2 radius massive explosion). the unit loses all charges upon discharging
- tesla tower unit, that can be charged, and it gets more range and damage the more charges it has (max 3). it has up to 5 range. it loses all charges upon firing.
- drone unit that can be charged. the more charges, the larger the movement (up to 2 charges so 3 movement). it can fly and it deals decent damage. it loses 1 charge per move.
- railgunner unit replacing catapult that can only attack in an 8-way pattern. it has 3 range, a lot of damage, and can pierce through enemies.
- jumpspark unit replacing the kight that can jump on enemies to make them conductive. conductive enemies release an explosion that also spreads the conductive effect, effectively making a chain reaction. currently cant attack tho for some reason.
- archeologist unit replacing mindbender that can excavate ruins. the ancients start with one but they dont start with its tech, kinda like shaman.
- ruins now give secret techs. there are 5 secret techs, and each ruin gives you a choice between 2. techs are: high voltage (unlocking tesla tower), charge storage (unlocking accumulator), aviation (unlocking drone), redirection (unlocking sentry) and charge logistics (unlocking sapper)

# Not implemented, but I have a complete plan:
- make battery produce 2 free pop but get an improvement effect after getting struck, and if it gets struck with lightning while having the effect, it explodes dealing massive damage and decreasing pop by 2. the effect wears down after 1 turn, meaning the player needs consistent charge logistics going or the battery explodes.
- sentry unit that redirects incoming damage from nearby units to it. it would be quite tanky, with 15-20hp and 2-3 defence. it could store 1 charge, and if it redirects while charged, it doesnt take damage but instead loses the charge.
- sapper unit with 1 max charge, that has sneak and creep, and can hit capacitor (=chargeable) units when charged to fully charge them. it would have dash and possibly escape. this unit could transfer charges from lightningrods to teslas or accumulators to use them more effectively, while keeping batteries going, or could go into enemy territory to sabotage lightning economy in mirror matches.
- overheat ability that damages units for 4 damage when shooting. if the unit has capacitor, the unit only loses health upon firing at max charge. railgunner and tesla will get this
- daggers get replaced by some unit that grows that same turn into some weaker dagger, but the unit can choose to rather kill itself and spawn a lightning strike on top of it.
